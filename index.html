안드로이드(java, kotlin)

개발 과정 및 상세 기술

  • jsoup 라이브러리를 이용하여 데이터 구문분석
  • zxing 라이브러리를 사용하여 QR코드 처리
  • firebase(FCM)를 이용하여 당첨 번호 알림
  • 로또 판매점 위치를 보여주기 위해 구글맵 사용
  • 서버컴(Ubuntu server)에 있는 데이터베이스 정보 데이터 통신을 위한 웹 제작
  • 웹 제작은 스프링 프레임워크와 mybatis 라이브러리 이용
  • 패턴 웹(MVC), 안드로이드(MVP, MVVM)
  • DB 웹 초기에는 Oracle로 개발하다 라이선스 문제 있을까 봐 Mysql > Mariadb로 정착
  • DB 안드로이드 초기에는 선택지가 없어 SQlite로 개발하다. 그 후에는 Room과 Realm의 성능 비교 후 수정 작업이 많으면 Room 수정보다는 select 중점일 경우에는 Realm을 사용
  • 안드로이드 서버컴 데이터베이스 정보를 받기 위해 초기에는 선택지가 없어 Asynctask에 HttpURLConnection과 스트림을 사용하여 정형적인 http통신을 사용
  • 후에 livedata + retrofit + databinding 또는 rxjava + databinding 방식으로 설계
